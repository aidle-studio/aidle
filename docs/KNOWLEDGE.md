# KNOWLEDGE.md

本ドキュメントは、プロジェクト固有の「暗黙知」「バグの教訓」「特殊な仕様の理由」を蓄積する構造化された記憶庫です。
AIエージェントは、**バグを修正した際や、重要な技術的決定（ADRにするほどではないもの）を行った際に、必ず本ファイルに追記（[x] 完了条件の一部）** してください。

## 🎯 運用ルール

- **形式の統一**: 新しい知見は必ず以下の「トラブルシューティング形式」または「Q&A形式」で追記すること。
- **検索性の確保**: 関連するキーワードやエラーメッセージをそのまま記載し、後でgrepしやすくすること。
