# Product Specs (SDD)

本ディレクトリは、製品仕様書（Product Specs）および受け入れ基準（Acceptance Criteria）を管理します。
AI駆動開発において、これらは「AIへの詳細な指示書」として機能します。

## 📋 仕様書一覧

- **[Specification Template](specification-template.md)**: 新しい仕様書を作成する際のテンプレート。
- **[Feature A](feature-a.md)**: 具体的な機能の仕様書（例）。

## 📝 仕様策定の原則

AIエージェントは、実装を開始する前に、必ず本ディレクトリ配下に仕様書を作成または更新し、人間と合意してください。

- **REQ-***: 機能要件を ID 付きで定義せよ。
- **AC-***: 各要件の受け入れ基準を ID 付きで定義せよ。
- **SDD**: 仕様駆動開発を徹底し、コードの前に「振る舞い」を確定させよ。
