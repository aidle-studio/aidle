# 🎤 aidle (アイドル)

> **Ready. Set. AI.** 🚀  
> AI駆動開発のベストプラクティスを詰め込んだプロジェクト雛形生成ツール。

`aidle` は、CursorやWindsurfなどのAIエディタがプロジェクトの文脈（Context）を即座に理解し、高精度なコード生成を行えるように設計されたボイラープレート・ジェネレーターです。

開発の初動で発生する「AIへのルール説明」や「環境設定」の手間をゼロにし、スムーズな開発の立ち上がり（アイドリング）を支援します。

---

## ✨ 主な機能

* **📂 AI-Optimized Structure**: AIがディレクトリ構成や技術スタックを誤認しないための最適なドキュメント配置。
* **🌱 Context Seeding**: プロジェクト独自の命名規則や設計方針をAIに定着させるための `CONTEXT.md` テンプレート。
* **💉 Best Practice Injection**: AIがバグを出しにくい「クリーンな骨組み」を、言語・フレームワークごとに生成。
* **🤖 Personality Config**: AIに特定の役割（シニア設計者、テスト重視など）を付与する設定ファイルの同梱。

---

## 🚀 クイックスタート

`aidle` を使って、新しいプロジェクトの「ステージ」をセットアップしましょう。

## 📁 生成されるプロジェクト構造

AI駆動開発（AIDD）を円滑にするための特別なファイルが含まれます。

## 💡 なぜ "aidle" なのか？

AI駆動開発において、生成されるコードの質は「与えられた文脈（Context）」の質で決まります。
aidle は、人間が手動で行っていた「文脈の整理」をテンプレート化。
プロジェクト開始直後から、AIがあなたの専属プロデューサーとしてフルパフォーマンスを発揮できる状態を作ります。
