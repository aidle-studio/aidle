# QUALITY_SCORE.md

本ドキュメントは、プロジェクトの品質基準と現在の「品質スコア」を可視化するものです。
AIエージェントは、実装の完了時にこの品質基準を満たしているかを確認してください。

## 🎯 品質基準 (Quality Standards)

| 指標 | 目標値 | 検証方法 |
| :--- | :--- | :--- |
| **テスト成功率** | 100% | `cargo nextest run` 等 |
| **カバレッジ (全体)** | 80% 以上 | `cargo llvm-cov` 等 |
| **カバレッジ (差分)** | 90% 以上 | `check_coverage_gate.sh` 等 |
| **Lintエラー/警告** | ゼロ | `cargo clippy` 等 |
| **フォーマット違反** | ゼロ | `cargo fmt --check` 等 |

## 📊 現在の品質スコア

- **最終更新日**: 20XX-MM-DD
- **現在のステータス**: [🟢 GREEN / 🟡 YELLOW / 🔴 RED]
- **備考**: [例: 初期セットアップ完了]

## 🛠️ 継続的改善 (GC)

エージェントは、タスクの合間にバックグラウンドでリファクタリング（ガベージコレクション）を実施し、品質スコアの向上に努めてください。
古いパターンや陳腐化したドキュメントの修正も、品質向上の重要な一環です。
