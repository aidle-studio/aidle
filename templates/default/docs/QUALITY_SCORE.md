# QUALITY_SCORE.md

本ドキュメントは、プロジェクトの品質基準と現在の「品質スコア」を可視化するものです。
AIエージェントは、実装の完了時にこの品質基準を満たしているかを確認してください。

## 🎯 品質基準 (Quality Standards)

| 指標 | 目標値 | 検証方法 |
| :--- | :--- | :--- |
| **テスト成功率** | 100% | `cargo nextest run` 等 |
| **UT 充足率 (Branch)** | Tier別目標 (80-100%) | `cargo llvm-cov` 等 |
| **IT/E2E 充足率** | 全AC/全遷移網羅 | `docs/TEST_PLAN.md` の確認 |
| **Live 健全性** | SLO 達成 / スモーク成功 | モニタリング結果の確認 |
| **AC 充足率 (密度)** | ACあたり平均 2.0以上推奨 | `docs/TEST_PLAN.md` のマトリクス |
| **Lintエラー/警告** | ゼロ | `cargo clippy` 等 |
| **フォーマット違反** | ゼロ | `cargo fmt --check` 等 |
