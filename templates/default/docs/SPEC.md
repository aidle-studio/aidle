# SPEC.md

本ドキュメントは、プロジェクトの機能要件と仕様を管理する **「製品仕様書（Source of Truth）」** です。
AI駆動開発（AIDD）において、本ファイルは「AIへの詳細な指示書」として機能します。
実装を開始する前に、必ず「要求（REQ）」と「受け入れ基準（AC）」を定義し、人間と合意してください。
