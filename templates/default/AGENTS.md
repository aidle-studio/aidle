# AGENTS.md

このプロジェクトは **エージェントファースト（Agent-First）** な開発環境です。
AIエージェント（Claude Code, Cursor, Windsurf等）は、このファイルを起点にプロジェクトを理解してください。

## 🗺️ プロジェクトの地図

詳細なルールやコンテキストは `docs/` 配下に「段階的開示」の原則で配置されています。
エージェントは、まず以下のファイルを順に読み、現在の状況を把握してください。

1. **[ARCHITECTURE.md](ARCHITECTURE.md)**: プロジェクトの全体構造、依存関係、不変の地図。
2. **[docs/HARNESS.md](docs/HARNESS.md)**: 【重要】セッション開始時に必ず確認し、自分のための検証レールを設営してください。
3. **[docs/AGENT_CONTEXT.md](docs/AGENT_CONTEXT.md)**: 現在のフェーズ、直近ゴール、次アクションの動的コンテキスト。
4. **[docs/RULES.md](docs/RULES.md)**: 開発プロセスと完了条件（DoD）の定義。

## 🚀 開発の進め方

本プロジェクトは **「計画 ➔ ハーネス ➔ 実装 ➔ 検証」** のサイクルで進行します。

- 自然言語の指示よりも、機械的な検証（Lint, Test）を優先してください。
- 詳細は `docs/` 配下の各専門ドキュメント（SPEC.md, TEST_PLAN.md等）を参照してください。

## 🤖 初期セットアップ

新しくプロジェクトに参画した際、または環境が変わった際は、直ちに **`docs/HARNESS.md`** に従って自分自身の検証環境（フック、リンター、検証スクリプト）をセットアップまたは確認してください。
