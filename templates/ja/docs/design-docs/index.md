## 🏛️ 設計ドキュメント一覧

- **[Core Beliefs](core-beliefs.md)**: プロジェクトが大切にする設計思想と信念。
- **[System Architecture](../ARCHITECTURE.md)**: プロジェクトの全体構造。
- **[Database Design](database.md)**: データベース設計（任意）。

## 📝 作成指針

AIエージェントは、複雑なコンポーネントや新機能を実装する前に、必要に応じてここに設計ドキュメントを作成し、人間と合意してください。
設計思想を明文化することで、AIによる将来の実装の一貫性を高めます。
