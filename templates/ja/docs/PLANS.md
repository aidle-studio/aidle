# PLANS.md

本ドキュメントは、プロジェクトのマクロなロードマップ、全体の進捗、および重要なフェーズ計画を定義します。

## 🗺️ ロードマップ (Roadmap)

プロジェクトの全体像と、各フェーズの目標を記述します。

| フェーズ | 目標 | ステータス |
| :--- | :--- | :--- |
| **P1: Foundation** | 基盤機能、品質ゲートの確立 | [🟢 完了 / 🟡 実行中] |
| **P2: Feature** | 主要機能の実装、UXの向上 | [⚪️ 未着手] |
| **P3: Release** | 最適化、セキュリティ、公開準備 | [⚪️ 未着手] |

## 📝 実行プランの運用

詳細な実行プランは、以下のディレクトリ配下に作成されます。

- **`docs/exec-plans/active/`**: 現在実行中の詳細プラン。
- **`docs/exec-plans/completed/`**: 過去に完了したプランのアーカイブ。

## 🚨 技術負債 (Technical Debt)

解消が必要な技術的負債、後回しにされたタスクのリストは `docs/exec-plans/tech-debt.md` で管理されます。
エージェントは、余裕があるときにこれらの解消を検討してください。
