## 🤖 AI駆動開発 (AIDD) について

本プロジェクトは **AI駆動開発 (AI-Driven Development)** のベストプラクティスを取り入れた構成になっています。
Copilot, Cursor, Windsurf, Gemini, Claude などのAIエディタ/エージェントが、プロジェクトの文脈を正確に理解し、自律的に開発を進められるように設計されています。

### 開発に参加するエージェント（AI）へ
新しくプロジェクトに参加した際は、まずルートディレクトリにある **`AGENTS.md`** を最初に読み込んでください。
そこから「Core Context」と呼ばれる基盤ドキュメント群（`ARCHITECTURE.md`, `RULES.md` 等）へと誘導され、安全かつ効率的な開発に必要なルールと現在地を把握することができます。
