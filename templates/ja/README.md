## 🤖 AI駆動開発 (AIDD) について

本プロジェクトは **AI駆動開発 (AI-Driven Development)** のベストプラクティスを取り入れた構成になっています。
Copilot, Cursor, Windsurf, Gemini, Claude などのAIエディタ/エージェントが、プロジェクトの文脈を正確に理解できるように設計されています。

新しいエージェントがプロジェクトに参加した際は、まずルートディレクトリにある **`AGENTS.md`** を読ませてください。そこからプロジェクトのルールや仕様（`docs/` 配下のファイル）へと誘導され、安全かつ効率的な開発がスタートします。
